## 快捷键设置

Speekium 桌面应用支持自定义快捷键，让你以最舒适的方式控制语音识别。

### 按键通话 (Push-to-Talk)

**默认快捷键**: `Alt+3` (跨平台统一)

按住此快捷键开始语音识别，松开时停止。

#### 如何自定义快捷键

1. 打开 Speekium 应用
2. 点击设置图标
3. 选择"快捷键"分类
4. 在"按键通话"旁边点击"录制"按钮
5. 按下你想要的快捷键组合
6. 设置会自动保存

#### 支持的快捷键格式

- **简单组合**: 最多 2 个修饰键 + 1 个字符键
- **修饰键**:
  - `⌘` (macOS) / `Ctrl` (Windows/Linux)
  - `⇧` / `Shift`
  - `⌥` / `Alt`
- **示例**:
  - `⌘2` - Cmd + 2
  - `⌘⇧V` - Cmd + Shift + V
  - `Ctrl+Alt+P` - Ctrl + Alt + P

#### 取消录制

- 按 `ESC` 键取消录制
- 只按修饰键不会保存（需要同时按字符键）

### 显示/隐藏窗口

**快捷键**: `⌘⇧Space` (Cmd + Shift + Space)

此快捷键用于全局显示或隐藏主窗口，目前暂不支持自定义。

## English Keyboard Shortcuts

Speekium desktop application supports customizable keyboard shortcuts for voice control.

### Push-to-Talk

**Default Shortcut**: `Alt+3` (Cross-platform)

Hold to start voice recognition, release to stop.

#### How to Customize

1. Open Speekium app
2. Click settings icon
3. Select "Shortcuts" category
4. Click "Record" button next to "Push-to-Talk"
5. Press your desired key combination
6. Settings auto-save

#### Supported Formats

- **Simple combinations**: Max 2 modifiers + 1 character key
- **Modifiers**:
  - `⌘` (macOS) / `Ctrl` (Windows/Linux)
  - `⇧` / `Shift`
  - `⌥` / `Alt`
- **Examples**:
  - `⌘2` - Cmd + 2
  - `⌘⇧V` - Cmd + Shift + V
  - `Ctrl+Alt+P` - Ctrl + Alt + P

#### Cancel Recording

- Press `ESC` to cancel recording
- Pressing only modifiers won't save (need character key too)

### Show/Hide Window

**Shortcut**: `⌘⇧Space` (Cmd + Shift + Space)

Global shortcut to show/hide main window. Currently not customizable.
